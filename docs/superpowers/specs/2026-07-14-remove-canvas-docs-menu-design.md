# 删除画布文档菜单设计

## 目标

删除画布左上角菜单中的“文档”入口，使用户无法再从该菜单跳转到 `https://docs.canvas.best/`。

## 修改范围

- 从画布页面的 Ant Design `Dropdown` 菜单项中删除“文档”。
- 删除画布页面不再使用的 `BookOpen` 图标和 `DOCS_URL` 导入。
- 删除全项目不再使用的 `DOCS_URL` 常量。
- 不调整其他画布菜单项、顶部导航或文档站点本身。

## 状态与异常处理

该入口是静态菜单项，不涉及状态、数据请求或异常处理。删除后不增加替代交互。

## 验证

手动进入任意画布，点击左上角菜单按钮，确认菜单中不再显示“文档”，其他菜单项仍正常显示和操作。
