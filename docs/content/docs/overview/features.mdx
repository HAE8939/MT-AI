# 功能介绍

MT-AI 主导航为五个模块：**画布 / 工作流 / 灵感广场 / 我的 / 配置**。本文档按模块记录当前已实现的主要功能。

## 画布

创作主战场，画布节点是唯一生成入口（独立的生图 / 视频工作台已并入画布）。

### 画布与项目

- 多画布项目：创建、重命名、删除、批量管理；单项目 zip 导入导出。
- 视图：拖动平移、滚轮缩放、缩放滑杆、重置视图、小地图定位；点阵 / 网格线 / 空白三种背景；浅色与深色主题。
- 选择与编辑：框选、多选、全选、复制粘贴（含连线）、`Ctrl/Cmd+D` 快捷复制、撤销重做（覆盖节点、连线、视口、背景与助手会话）。
- 「整理画布」一键将图片节点按网格排列并定位视图，保留连线与分组归属。
- 多选图片支持「批量下载」（打包 zip）和「宫格拼合」（可设行列、间距、背景色，合成为新节点并与原图连线）。

### 节点

六类节点，自由连线组织生成流程：

| 节点 | 用途 |
| --- | --- |
| 文本 | 提示词、说明文案、AI 文字回答 |
| 图片 | 上传图、生成结果、素材图；可标注参考用途（风格 / 构图 / 色彩 / 材质），生成时以 `[图片N: 风格参考]` 形式注入提示词 |
| 视频 | 上传或生成的视频，原生播放器预览 |
| 音频 | TTS 生成结果或上传音频 |
| 生成配置 | 汇总上游文本 / 图片 / 视频 / 音频，统一配置模型、比例、数量后批量生成 |
| 组 | 拖入吸附成组、拖出解除，拖动组整体移动子节点 |

节点通用能力：拖拽移动、四角缩放、命名（默认名显示在节点上方，双击编辑）、查看信息与 JSON、左右连接点建立上下游关系。批量生成结果先展示为图片组节点，支持叠卡预览、展开与设置主图。

### AI 生成

- 文生图、图生图、参考图编辑、文本问答、视频生成、TTS 音频，全部在画布节点上完成。
- 渠道支持 OpenAI / Gemini 两种调用格式、东木-AI 聚合平台（模型与能力自动发现）、火山方舟 Seedance；详见[配置指南](/docs/canvas/config-guide)。
- 视频生成可同时读取上游文本（prompt）、图片（参考图）、视频（参考视频）、音频（参考音频）；Seedance 2.0 支持最多 9 张参考图、3 个参考视频、3 个参考音频，分辨率 `480p/720p/1080p`（fast 模型不支持 1080p），比例含 `21:9` 与 `adaptive`，时长 4-15 秒或智能时长。
- 生成失败可重试；每个生成节点在 `metadata` 中记录提示词、模型、尺寸与参考图引用，重试时参考图丢失会直接提示。

### 图片工具箱

- 双图滑杆对比（弹窗可按 `F` 全屏切换）。
- 360° 全景查看与生成（固定单次 2:1 配置）。
- 非破坏性标注：文字 / 画笔 / 橡皮擦，导出时才合成进图片，可「确认并裁剪」直连裁剪流程。
- 裁剪（自由 / 1:1 / 4:3 / 3:4 / 16:9 / 9:16 预设）与行列切图（切图线可拖拽、增删、重置）。
- 局部重绘：矩形比例选区（8 向拉杆、像素尺寸标签）或自由画笔蒙版、羽化半径、附加参考图；OpenAI 与 Gemini 两种调用格式均可用，选区元数据随结果保存便于失败重试。
- 多角度生成（three.js 相机预览的通用模型模式）与反推提示词。
- 图片放大（本地插值）。

### 侧栏三合一

画布右侧面板固定三个 tab：

- **对话**：项目模型 / 本地 Codex 双引擎。项目模型走已配置渠道；本地 Codex 通过 Canvas Agent 连接，可直接操作画布（见 [Codex 能力清单](/docs/overview/codex-capabilities)）。
- **工作流**：RunningHub 式运行面板，云工作流与本地工作流都在此填参数、运行并跟踪状态，结果写回画布节点。
- **提示词**：浏览 / 搜索 / 分组提示词库，常驻「组合模板」分组，点选标签实时组合 JSON 提示词，「填入选中节点」一键写入。

### 文档智能体

内置可增删改的分析角色（`web/public/roles.json` 自动转换为内置模板）：读取选中的图片 / 文本节点，用角色 System Prompt 生成连接的分析文本节点，结果节点可重新生成。分析结果里的 JSON 可一键「结构化」为侧栏可点选卡片，微调后回填或保存到提示词库。

### 任务中心

画布 AI 生成统一进入任务中心：

- 任务卡展示结果缩略图，支持全屏预览、下载、复制图片与链接、带提示词的图文合成下载。
- 「定位」一键跳转到任务对应的画布节点（跨项目自动切换画布），节点被删也能从任务中心找回结果。
- 进度条（轮询响应带进度时）或已等待时长展示；取消、失败重试、清除已完成、刷新恢复。

## 工作流

三种来源，统一在画布侧栏或 `/workflows` 页运行，详见[工作流使用指南](/docs/canvas/workflows-guide)：

- **RunningHub 云工作流**：登记 workflowId + 参数映射，粘贴「导出工作流 API」JSON 自动识别参数；接口走 OpenAPI v2（Bearer 鉴权），内置「Z Image 亿级像素文生图」模板开箱即用。
- **画布模板**：画布多选节点「保存为工作流模板」，保留生成参数骨架，插入任意画布换输入重跑。
- **本地工作流**：画布多选生成步骤「保存为本地工作流」并标记输入槽，填素材后按依赖顺序自动串跑本地生成（文生图 → 参考图编辑 → …），不依赖云端（v1 支持图片链）。

## 灵感广场

- 100+ 条室内行业提示词（SU 转写实、室内效果图、建筑外观、景观规划、商业空间等九大类），搜索 / 标签筛选 / 一键收藏。
- 条目只有「复制提示词」和「收藏」两个动作；收藏落入「我的 → 收藏」，提示词到画布统一为复制 → 粘贴（或经侧栏「提示词」tab 直填）。
- 「组合模板」分类收录 DMDS 式提示词生成器：卡片 → 键 → 标签三级结构，含「室内效果图组合工作台」六张卡片（空间类型 / 核心约束 / 场景与光效 / 材质控制 / 摄影参数 / 渲染精度），嵌套 JSON 实时预览。
- 第三方数据来源与授权见仓库 [NOTICE.md](https://github.com/HAE8939/MT-AI/blob/main/NOTICE.md)。

## 我的

四个分区：**收藏 / 我的提示词 / 素材 / 生成记录**。

- **收藏**：从灵感广场收藏的条目，可取消收藏，自动去重。
- **我的提示词**：新建、编辑、分组管理自建提示词；编辑器支持多卡片草稿（`*标签=实际值` 行语法）、任意 JSON 一键导入成卡片、「AI 增强」把简单描述改写为专业结构化提示词；全量 JSON 导入导出。
- **素材**：批量 / 拖拽导入（含文件夹），入库自动生成缩略图；编辑标题、封面、标签、来源、备注；图片素材可挂载提示词，插入画布时随数据带出；搜索、筛选、分页、批量删除。
- **生成记录**：统一任务运行时的历史视图，缩略图、状态、定位画布节点、失败重试。

## 配置

集中一页管理，六个分区：渠道 / 模型 / 偏好 / 腾讯云 COS / RunningHub / Codex。多渠道管理（OpenAI 兼容与 Gemini 调用格式自选、拉取模型列表、一键添加东木-AI 聚合平台）、生成偏好、COS 媒体同步、本地 Codex Agent 连接。细节见[配置指南](/docs/canvas/config-guide)。

## 数据与限制

- 画布 / 提示词 / 素材 / 生成记录存浏览器 IndexedDB（localforage）；媒体文件可选腾讯云 COS 本地优先上传队列（自动重试 + 同步中心）。
- `config.json` 支持管理员级配置下发（渠道、COS、RunningHub），团队部署时自动同步到所有设备。
- API Key 保存在浏览器本地并由浏览器直连接口，只适合个人或可信环境使用。
- Seedance 本地参考视频 / 音频建议使用公网可访问 URL；远程视频 URL 会尽量下载为本地 Blob 持久化，受 CORS 限制时保留远程 URL。
- 画布更适合桌面端使用，移动端触控体验还未系统完善。
- 项目处于快速开发阶段，本地存储格式可能直接调整、不保证历史数据兼容。
