# MT-AI 提示词引擎工作流使用指南

## 什么是提示词引擎工作流

提示词引擎工作流是 MT-AI 的一项强大功能，它将室内设计领域的专业知识沉淀为 JSON 配置，通过 LLM（大语言模型）自动将用户的简单描述扩写为专业的英文提示词，再驱动 AI 图像模型生成高质量效果图。

**核心理念**：
- 用户只需说一句大白话（如"把墙换成微水泥，浅色"）
- LLM 自动扩写为专业提示词（包含材质描述、光影保留、防翻车短语）
- 直接生成专业效果图，无需手写复杂提示词

## 22 个可用工作流

### 🎭 蒙版编辑类（6 个）

需要涂抹蒙版指定修改区域：

| 工作流 | 用途 | 输入 |
|--------|------|------|
| **场景添加人物** | 在室内场景中自然添加人物 | 原图 + 蒙版 + 人物描述 |
| **局部开灯** | 模拟灯具开启后的光照效果 | 原图 + 蒙版（灯具+照射范围）|
| **局部材质修改** | 按材料名称修改表面材质 | 原图 + 蒙版 + 材料名 |
| **指定人物生成** | 按参考图指定人物外观 | 原图 + 蒙版 + 人物参考图 |
| **指定材质替换** | 按参考图替换表面材质 | 原图 + 蒙版 + 材质参考图 |
| **软硬装局部替换** | 蒙版区域替换家具/装饰 | 原图 + 蒙版 + 物体描述 |

**使用技巧**：
- 涂抹范围 = 修改范围
- 替换物体要连同影子一起涂抹
- 局部开灯：把灯具 + 想被照亮的范围一起涂抹

### 🌅 全图编辑类（14 个）

处理整张图片，不需要蒙版：

| 工作流 | 用途 | 输入 |
|--------|------|------|
| **光影大师** | 按参考图重塑光影 | 原图 + 光影参考图 |
| **光影重塑** | 一键去灰提亮优化光影 | 原图 + 天气描述（可选）|
| **室内氛围转换** | 迁移参考图光影氛围 | 原图 + 氛围参考图 |
| **室内风格转化** | 迁移参考图设计风格 | 原图 + 风格参考图 |
| **白膜出图** | 3D 白模渲染为效果图 | 白模图 + 风格参考图（可选）|
| **手绘线稿出图** | 手绘线稿转效果图 | 线稿图 + 风格参考图（可选）|
| **毛坯房出图** | 毛坯房装修效果图 | 毛坯图 + 风格描述 |
| **软装拼贴出图** | 物料拼贴板转实景 | 拼贴板图 |
| **质感增强** | 提升照片真实感 | 原图 + 强度档位 |
| **全景渲染** | 生成 2:1 全景图 | 原图 |
| **一键彩平图** | 平面布置图转彩平 | 黑白平面图 + 参考图（可选）|
| **一键材质通道图** | 生成材质 ID Map | 效果图 |
| **软装物料清单整理** | 实景提取物料清单板 | 实景图 + 比例选择 |
| **家装平面方案生成** | 结构图转平面布置图 | 结构图 |

**推荐串联链路**：

1. **洗图三部曲**：光影重塑 → 质感增强 → 高清放大
2. **出图链路**：白膜/毛坯/手绘出图 → 光影重塑 → 质感增强 → 高清放大
3. **平面链路**：家装平面方案生成 → 一键彩平图
4. **软装闭环**：软装物料清单整理 ↔ 软装拼贴出图

### 📐 特殊类型（2 个）

| 工作流 | 用途 | 类型 |
|--------|------|------|

## 如何使用

### 1. 在画布侧栏访问

1. 打开画布项目
2. 点击右侧「工作流」标签
3. 选择「提示词引擎工作流」分类
4. 选择想要运行的工作流

### 2. 填写参数

根据工作流的 `inputSpec` 动态显示输入项：

- **上传图片**：点击上传或从画布节点选择
- **涂抹蒙版**：使用画笔涂抹需要修改的区域
- **参考图**：上传风格/材质/人物参考图
- **文本输入**：简单描述需求（一句话即可）
- **选项设置**：档位、分辨率等预设选项

### 3. 运行工作流

点击「立即生成」后：

1. **LLM 扩写提示词**（~5 秒）
   - 系统加载工作流的 JSON 知识库
   - LLM 看图并结合用户输入
   - 自动扩写为 60-140 词专业英文提示词

2. **图像模型生成**（~30-60 秒）
   - 调用配置的图像模型（如 gpt-image-1）
   - 按扩写的提示词生成图片

3. **结果写入画布**
   - 自动创建图片节点
   - 连接到源节点
   - 元数据保存 `final_prompt`（可查看）

### 4. 查看扩写提示词

- 右键点击生成的图片节点
- 选择「查看扩写提示词」
- 查看 LLM 自动生成的完整提示词

**为什么要查看**：
- 学习专业提示词的写法
- 调试生成效果（定位是扩写问题还是模型问题）
- 回流优化 JSON 配置

## 高级功能

### 1. 与本地工作流串联

可以将多个提示词引擎工作流组合为本地工作流：

1. 在画布上手动运行几个步骤（如：毛坯房出图 → 光影重塑 → 质感增强）
2. 选中所有相关节点
3. 点击「保存为本地工作流」
4. 标记输入槽（原始毛坯图 + 风格描述）
5. 下次直接运行，自动串跑所有步骤

### 2. 与 Canvas Agent 联动

如果安装了本地 Canvas Agent（Codex/Claude Code），可以对话触发：

```
你: "帮我用'局部材质修改'把这张图的墙面换成微水泥"

Agent: [自动定位工作流 → 填充参数 → 运行 → 结果写入画布]
```

### 3. 自定义工作流（待实现）

## 技术原理

### JSON 知识库结构

每个工作流的 JSON 包含三层：

```json
{
  "meta": {
    "id": "workflow-id",
    "name": "工作流名称",
    "taskType": "masked-edit | full-edit | generate | upscale"
  },
  "inputSpec": {
    "image": "required | optional | none",
    "mask": "required | optional | none",
    "userText": "required | optional | none"
  },
  "promptEngine": {
    "llmGuidance": { /* 硬契约 vs 脚手架 */ },
    "coreFormula": { /* 扩写公式 */ },
    "outputContract": { /* 英文/单段/长度 */ },
    "templates": [ /* few-shot 示例 */ ],
    "failureModes": [ /* 防翻车短语 */ ]
  }
}
```

### 双层效力设计

**硬契约**（必须严格遵守）：
- `outputContract`：输出格式要求（英文、单段、长度上限）
- `maskProtocol`：蒙版权威关系（透明区 = 重绘区）
- `failureModes`：防错语义（防止悬浮、纹理失真等常见翻车）

**脚手架**（质量标杆，鼓励超越）：
- `templates`：few-shot 示例，展示合格写法
- 词汇库：材质库、风格库等（开放示例，非封闭选项）
- 规则原则：引导 LLM 自行判断，非机械执行

### 为什么需要 LLM 扩写

直接原因：AI 图像模型（gpt-image-1）只听得懂高质量英文提示词

用户说："把墙换成微水泥，浅色"

LLM 扩写后：
> "Change the sofa feature wall in the masked area to seamless microcement in a light warm grey with a subtle trowel texture and matte finish. Keep the wall's geometry and outline unchanged, preserving the existing artwork shadows and the wall-washing light gradient on the new surface. The trowel texture stays fine and understated. Do not alter anything outside the edited area."

LLM 自动完成了：
- 中文材料名 → 完整英文材质描述
- "浅色" → light warm grey
- 自动补充：保留挂画阴影、保留洗墙光、纹理细腻、越界保护

这些专业知识全部来自 JSON 知识库。

## 成本说明

每次运行工作流 = 1 次 LLM 扩写 + 1 次图像生成

- **LLM 扩写**：~2k tokens，约 $0.002-0.01（取决于模型）
- **图像生成**：按您配置的渠道计费
- **总成本**：约 $0.05-0.20 / 次

**优化建议**：
- 在配置页选择性价比高的扩写模型（如 GPT-4o-mini）
- 简单工作流可跳过扩写，直接用模板（待实现）

## 常见问题

### Q: 为什么生成效果不理想？

1. **先查看 `final_prompt`**：
   - 右键图片节点 → 查看扩写提示词
   - 判断是扩写问题（提示词本身写得不好）还是模型问题

2. **扩写问题**：
   - 改进用户输入，提供更具体的描述
   - 反馈给开发团队，优化 JSON 配置

3. **模型问题**：
   - 尝试其他模型或调整参数
   - 某些效果可能是模型能力限制

### Q: 蒙版类工作流边缘有断层怎么办？

- 确认蒙版涂抹完整（包括物体影子）
- 检查蒙版与原图分辨率是否一致
- 可能是蒙版 alpha 转换问题，反馈给开发团队

### Q: 可以调整扩写的提示词吗？

目前不支持直接编辑。但您可以：
- 查看 `final_prompt` 学习专业写法
- 调整用户输入，影响扩写结果
- 未来版本将支持「查看并编辑提示词」功能

### Q: 高清放大功能为什么提示"开发中"？

高清放大不走生成式模型（会重绘内容），需要接入专业放大服务（Real-ESRGAN / Topaz 等）。该功能计划在后续版本实现。

临时方案：下载图片后使用第三方放大服务。
